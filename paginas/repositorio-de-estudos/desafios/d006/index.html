Tags HTML5

Em HTML5, podemos utilizar uma série de tags.

Para utilizar uma tag deve haver um objetivo, vou listar alguns e vou inserir na descrição a tag, para isso é só passar o mouse em cima para saber qual tag utilizar.

Para fazer o desafio proposto eu vou usar a tag para abreviações, pois é a unica forma que eu conheço para chegar no objetivo, mais para frente se eu descobrir outra forma irei corrigir.

A duas formas de fazer esse desafio:

  1. A primeira qual usei de forma errada pois era a unica ferramenta de meu conhecimento, pois é um elemento de abreviação <abbr> então seria melho eu colocar da seguinte forma:
    • <!DOCTYPE html>
    • <p>
  2. A segunda será qual urarei para complementar esse desafio, é um atributo title= dentro do elemento <p> qual se adequa melhor ao desafio proposto.
